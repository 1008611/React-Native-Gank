## Usage (iOS版本可以跑通，Android正在努力适配中)  
1. 安装以及正确调试React-Native开发环境（支持Windows／OSX）
2. clone本项目之后，在根目录下使用`npm install` 命令来安装项目中依赖的所有依赖库
3. 点开ios文件夹下的`ReactNativeGank.xcodeproj`运行程序
